Question-1: What are the different ways to select an element in the DOM?
There are many ways to select an element in the DOM.
  1. getElementById(): Select element by its id
  2. getElementsByClassName(): Select element by class name
  3. getElementsByTagName(): Select element by tag name
  4. querySelector():Selects the first matching elements
  5. querySelectorAll():Select all matching element

Question-3: What is event delegation in the DOM?
Event delegation is a technic where we set event listener in parent element and event handle their child element.
Question-4: What is event bubbling in the DOM?
Event bubbling is a js mechanism. If you click on child element, the element will go up to its parent,then grandparent and finally to the document
Question-5: How do you create, add, and remove elements using JavaScript?
For create an element with js we can use document.createElement().and then we want to add this element we can use appendChild() or append(), and lastly if we want to remove a element we can use removeChild() or remove().
